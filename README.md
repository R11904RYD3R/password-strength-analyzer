# Password Strength Analyzer

## Project Overview

The Password Strength Analyzer is a Python-based security automation project designed to evaluate the strength of a password based on common password security best practices. The program helps users identify weak passwords and provides suggestions for creating stronger passwords.

## Features

* Checks password length
* Checks for uppercase letters
* Checks for lowercase letters
* Checks for numbers
* Checks for special characters
* Calculates a password strength score
* Classifies passwords as Weak, Medium, or Strong
* Detects commonly used weak passwords
* Provides suggestions for improving password security
* Generates random passwords upon user request

## Requirements

* Python 3.14 or later
* Visual Studio Code (recommended)

## How to Run the Program

1. Download or clone the repository.
2. Open the project folder in Visual Studio Code.
3. Open a terminal.
4. Run the following command:

```bash
py password_analyzer.py
```

5. Enter a password when prompted.
6. Review the password strength results and recommendations.
7. Optionally generate a strong password.
